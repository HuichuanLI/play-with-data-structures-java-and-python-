
# 红黑树的定义
## 1．每个节点或者是红色的，或者是黑色的 
## 2. 根节点是黑色的 
## 3. 每一个叶子节点（最后的空节点）是黑色的 
## 4．如果一个节点是红色的，那么他的孩子节点都是黑色的 
## 5.从任意一个节点到叶子节点，经过的黑色节点是一样的
